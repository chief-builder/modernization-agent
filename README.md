# Modernization Agent

An autonomous multi-agent system for codebase modernization, built on Claude Code.

## Overview

The Modernization Agent helps maintain, refactor, and modernize existing codebases through four specialized modes:

1. **Discovery Mode**: Analyze and document codebase functionality
2. **Coverage Mode**: Ensure comprehensive test coverage
3. **Enhancement Mode**: Apply enhancements safely
4. **Migration Mode**: Perform tech stack rewrites with behavioral validation

## Usage

```bash
modernize discover /path/to/project      # Analyze codebase
modernize coverage /path/to/project      # Improve test coverage
modernize enhance /path/to/project       # Apply enhancements
modernize migrate /path/to/project       # Migrate tech stack
modernize status /path/to/project        # Check progress
